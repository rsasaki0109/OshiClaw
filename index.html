`OshiClaw` の見た目と会話テンポをそのまま触れる静的デモです。 GitHub Pages 上では backend を動かせないので、ここでは `無料の公式立ち絵 / 3D素材のあるキャラ` を中心に、1体だけ Live2D 実動作を混ぜて公開しています。

見どころ

  • ずんだもん、玄野武宏、雨晴はうなど無料素材の強いキャラを優先表示
  • 1体は Live2D 実動作、他キャラは fallback 見た目で切替
  • 会話のストリーミング風アニメーション
  • Markdown を含んだ返答表示
  • TTS とリアクション演出の一部をブラウザ内で再現

今の featured

  • ずんだもん: 公式3D / Live2D 導線が強い main face
  • 玄野武宏: 無料立ち絵と MMD モデルあり
  • 雨晴はう: 公式サイトで立ち絵や 3D 素材配布を案内
  • みお: この demo で実際に Live2D が動く確認用キャラ

ローカル版との差分

  • GitHub Pages では `/api/chat` を使わず、会話はプリセット再生です
  • モデル切替、履歴永続化、実 LLM 応答は repo のローカル版で利用できます
  • 画像素材や 3D 素材の導線は多いですが、実 Live2D モデル資産はまだ少ないです

ローカル起動

python3 -m pip install -r requirements.txt
cp .env.example .env
python3 server.py
