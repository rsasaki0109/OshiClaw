Static Demo
Pages 版は静的デモです。実際の LLM チャットはローカル実行版で動きます。

見どころ

  • みおは Live2D、他キャラは fallback 見た目で切替
  • 会話のストリーミング風アニメーション
  • Markdown を含んだ返答表示
  • TTS とリアクション演出の一部をブラウザ内で再現

ローカル版との差分

  • GitHub Pages では `/api/chat` を使わず、会話はプリセット再生です
  • モデル切替、履歴永続化、実 LLM 応答は repo のローカル版で利用できます
  • Live2D モデル資産は今のところ 1 体だけです

ローカル起動

python3 -m pip install -r requirements.txt
cp .env.example .env
python3 server.py
