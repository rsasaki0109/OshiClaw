同梱サンプル Live2D 実動作 Pages では bundled sample の Live2D だけを実体表示し、公式キャラは出典リンクだけを置いています。
Pages 版は静的デモです。実際の LLM チャット、キャラ切替、ローカルモデル運用はローカル実行版で動きます。

見どころ

  • bundled sample の Live2D を実際に動かして見せる
  • 公式キャラ候補は select から切り替えて出典リンクを確認できる
  • 会話のストリーミング風アニメーション
  • Markdown を含んだ返答表示
  • TTS とリアクション演出の一部をブラウザ内で再現

公式候補

  • ずんだもん: 公式ショップに VTubeStudio / 3D 配布あり
  • 東北ずん子: 公式ショップに VTubeStudio / 3D 配布あり
  • 玄野武宏: VirVox で MMD モデル配布あり
  • Pages では案内だけに留め、素材やモデル自体はミラーしていません

ローカル版との差分

  • GitHub Pages では `/api/chat` を使わず、会話はプリセット再生です
  • public ページには bundled sample 以外の素材やモデルを同梱していません
  • モデル切替、履歴永続化、実 LLM 応答は repo のローカル版で利用できます

ローカル起動

python3 -m pip install -r requirements.txt
cp .env.example .env
python3 server.py
